Desarrollo Web

Resumen

Esta página muestra mis habilidades en el diseño y desarrollo de sitios web responsivos y accesibles. Destaca mi capacidad para estructurar HTML limpio, crear CSS mantenible e implementar componentes interactivos en JavaScript como carruseles y menús de navegación. También integro sistemas de gestión de contenidos como WordPress para ofrecer portafolios escalables y profesionales.

Habilidades Destacadas

  • Desarrollo Frontend: HTML5, CSS3 (flexbox, grid, clamp para tipografía responsiva)
  • JavaScript: componentes interactivos (carrusel, navegación, funciones de accesibilidad)
  • Diseño Responsivo: layouts mobile-first, estructura semántica, buenas prácticas de accesibilidad
  • Branding & Estilo: estética minimalista, tipografía profesional, integración de logotipo
  • CMS: WordPress para gestión escalable de contenidos
  • Flujo de trabajo: CSS semántico, componentes reutilizables, accesibilidad (navegación por teclado, soporte para lectores de pantalla)
