Análisis de Datos

Resumen

Esta página muestra mis habilidades en el análisis de datos de movilidad urbana, extracción de insights y presentación clara de resultados. Destaca mi capacidad para manejar conjuntos de datos, aplicar métodos estadísticos y comunicar hallazgos de manera efectiva, demostrado en proyectos como el Análisis de Movilidad Urbana Ecobici 2024.

Habilidades Destacadas

  • Programación & Herramientas: R (tidyverse, lubridate, leaflet), Python (pandas, numpy, seaborn), SQL (DuckDB, MySQL, PostgreSQL, DB2)
  • Análisis Exploratorio de Datos: tendencias temporales, espaciales y demográficas
  • Preparación de Datos: limpieza, transformación, reportes reproducibles (R Markdown)
  • Visualización: Tableau, Power BI (básico), Matplotlib, Seaborn
  • Métodos Analíticos: análisis de cohortes, fundamentos de pronósticos, pruebas de hipótesis
  • Comunicación: storytelling empresarial y presentación de insights

Proyecto Destacado

Análisis de Movilidad Urbana Ecobici 2024

Un análisis en profundidad de datos de bicicletas compartidas para comprender patrones de uso, horarios pico y rendimiento de estaciones. Este proyecto incluyó la limpieza de grandes conjuntos de datos, análisis exploratorio y visualización de tendencias clave para apoyar decisiones de planificación urbana.
